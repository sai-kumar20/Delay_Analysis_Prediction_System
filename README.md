🎯 Why we built this system?

Steel plants generate a large amount of work delay data every day, making it difficult to analyze and identify the causes of delays manually.

Our system transforms this complex data into interactive dashboards with:

Shop-wise Analysis
Equipment-wise Analysis
Agency-wise Analysis
Duration-wise Analysis
Conveyor-wise Analysis
Season-wise Analysis

This helps employees and management understand delay patterns quickly, identify problem areas, and make better operational decisions.

The Machine Learning prediction module further helps by identifying equipment that is likely to experience delays, enabling preventive maintenance and reducing future work delays.

💡 What I worked on
Cleaned and processed 80,000+ real-world industrial records with uneven raw data.
Designed a MySQL database for efficient data management.
Built analysis dashboards and developed a Machine Learning prediction model.
Developed and deployed the complete full-stack web application.
